# C Programming

A collection of C programs and projects documenting my journey from learning the fundamentals of the C programming language to building engineering applications.

This repository is part of my Electrical and Computer Engineering learning roadmap. The goal is to develop strong programming fundamentals before moving on to embedded systems, FPGA development, digital signal processing (DSP), operating systems, and computer architecture.


## Objectives

* Master the C programming language
* Develop problem-solving and algorithmic thinking
* Learn low-level programming concepts
* Build a portfolio of progressively challenging projects
* Prepare for embedded systems and systems programming
